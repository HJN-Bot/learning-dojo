# learning-dojo

A long-term, public learning repo for **code + writing + skills**.

## How we learn (Feynman Coach Protocol)
For each topic, we follow a consistent flow:

0) **Intake** — goal, current level, and acceptance criteria
1) **Feynman Explain** — 3 layers (kid / engineer / boundaries)
2) **Socratic Deep Dive** — 5 follow-up questions
3) **Knowledge Check** — MCQs + short answers + 20-min practice
4) **Publish** — a 1-page reusable note (and optionally a small artifact)

See: `SKILL_FeynmanCoach.md`

## Repo structure
- `learning/topics/` — one folder per topic
- `code/` — small practice code
- `writing/` — essays/scripts/notes
- `templates/` — reusable templates
